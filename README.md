# uni_wave
Uni_Wave is an application for students and teachers from The University of Manchester. 

# Start Application Commands
## Start Server
nodemon app

## Start React Application 
npm start

## Start Socket
npm run socket

## Video Call Application 
### Start Video Call Application 
nodemon server.js

### Start Video Call
peerjs --port 3001
